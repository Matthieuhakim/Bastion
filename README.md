# Bastion

**Status: in development.**

Bastion is a Python SDK that sits at the tool-call boundary of an AI agent and does three things: enforces policies, escalates to a human when needed, and produces a tamper-evident signed record of every decision. It is open source, local-only, and has no server component.

The full README lands in Phase 12. Until then, see [Bastion_BuildPlan.md](./Bastion_BuildPlan.md) for the design and roadmap.
